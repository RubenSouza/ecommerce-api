## Api para Ecommerce

Este projeto é parte do curso do Matheus Machado "Criando um Ecommerce do Zero com NodeJS, React e MongoDB", aqui está a API que será responsável pelo funcionamento correto do Ecommerce.

### Tecnologias

- Node
- MongoDb
- Express
- Jwt
- Joi
